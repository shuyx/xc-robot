1 XC-ROBOT Web GUI 使用说明

1.1 概述

XC-ROBOT Web GUI 是基于 Qt + HTML 混合架构的现代化机器人控制界面，将原有的传统 Qt 控件替换为美观的 HTML5 界面，同时保持 Python 后端的稳定性。

1.2 功能特点

1.2.1 🎨 现代化界面设计

  • 渐变背景: 专业的蓝紫色渐变设计
  • 流畅动画: CSS3 过渡效果和悬停动画
  • 响应式布局: 自适应不同屏幕尺寸
  • 卡片式设计: 现代化的功能模块展示

1.2.2 🔧 技术架构

  • 前端: HTML5 + CSS3 + JavaScript
  • 后端: Python + PyQt5 + QWebEngineView
  • 通信: Qt Web Channel 双向通信
  • 渲染: Chrome 内核渲染引擎

1.2.3 🚀 主要功能

1.2.3.1 1. 设备连接测试

  • 右臂连接: FR3 右臂 (192.168.58.2)
  • 左臂连接: FR3 左臂 (192.168.58.3)
  • 底盘连接: Hermes 底盘 (192.168.31.211:1448)
  • 视觉系统: ToF相机和2D相机状态

1.2.3.2 2. 机械臂控制

  • 单臂控制（左臂/右臂）
  • 双臂协调控制
  • 实时状态监控

1.2.3.3 3. 底盘控制

  • 移动控制
  • 导航功能
  • 路径规划

1.2.3.4 4. 仿真系统

  • 机器人仿真
  • 3D 仿真环境
  • 任务编排

1.2.3.5 5. 系统监控

  • 实时日志显示
  • 系统状态监控
  • 性能统计

1.3 安装要求

1.3.1 基础依赖

pip install PyQt5 PyQtWebEngine PyYAML requests

1.3.2 系统要求

  • Python 3.6+
  • Qt 5.12+
  • Chrome/Chromium 内核支持

1.4 启动方式

1.4.1 方式1: 直接启动

python start_web_gui.py

1.4.2 方式2: 模块启动

from gui import get_web_main_window
XCRobotWebMainWindow = get_web_main_window()
window = XCRobotWebMainWindow()
window.show()

1.5 界面操作

1.5.1 连接测试

  1. 点击设备连接卡片
  2. 系统自动执行连接测试
  3. 实时显示连接状态和延迟

1.5.2 功能导航

  • 侧边栏: 功能模块分组导航
  • 搜索框: 快速查找功能
  • 收藏夹: 常用功能快速访问
  • 最近使用: 历史操作记录

1.5.3 日志系统

  • 实时日志: 系统操作实时记录
  • 日志级别: SUCCESS/INFO/WARNING/ERROR
  • 自动清理: 保持最新20条日志
  • 保存导出: 支持日志保存功能

1.6 快捷键

  • Ctrl+E: 紧急停止
  • F5: 重新加载界面
  • Ctrl+S: 保存日志

1.8 故障排除

1.8.1 常见问题

  1. PyQtWebEngine 未安装

    pip install PyQtWebEngine

  2. HTML 文件未找到

    • 检查 UI/xc_os_newui.html 文件是否存在
    • 确保文件路径正确

  3. 连接测试失败

    • 检查设备IP地址配置
    • 确认网络连接正常
    • 验证设备电源状态

  4. 界面显示异常

    • 按 F5 重新加载界面
    • 检查浏览器内核版本

1.8.2 调试模式

  • 按 F12 打开开发者工具（如果支持）
  • 查看控制台输出信息
  • 检查网络请求状态
